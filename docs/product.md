## 1. 项目背景与核心策略

### 1.1 项目背景

2026年（农历丙午马年）春节将至，年轻用户面临由于代沟产生的亲戚社交压力（催婚、问薪资），同时也存在强烈的节日娱乐、晒圈（朋友圈）需求。

### 1.2 核心价值

利用 AI 技术提供“情绪价值”和“社交货币”。通过“发疯/高情商回怼”解决社交恐惧，通过“搞笑存单”和“定制头像”提供社交谈资。

### 1.3 产品形态

- **平台：** 微信小程序（主战场） + H5（备用/站外引流）。
- **名字：** **春节嘴替**
- **Slogan：** 2026马年过年必备神器，你的全能社交互联网嘴替。

## 2. 功能模块与需求规格 (PRD)

本项目采用 **“1+2”** 聚合模式：1个核心引流功能 + 2个辅助变现/留存功能。

### 模块一：AI 嘴替（核心功能 - 流量引擎）

- **场景：** 用户模拟与亲戚聊天，AI 生成神回复。
- **评分：** 稀缺性 ★★★★ | 传播性 ★★★★★
- **功能逻辑：**
  1.  **角色选择：** 首页展示 4 个角色（势利眼二姨、催婚大姑、凡尔赛邻居、严厉二舅）。
  2.  **对话界面：** 仿微信聊天窗口。用户输入文本/语音。
  3.  **AI 生成：** 调用 DeepSeek API，基于 Persona（人设）生成回复。
  4.  **评价系统：** AI 回复后，自动附带“杀伤力评分”或“阴阳怪气指数”。
  5.  **裂变点：** “生成聊天长图”按钮，强制分享。
- **Prompt 核心设定（System）：**
  > 角色：中国式长辈。任务：对用户进行压迫式提问或反击。风格：阴阳怪气、过度热情或道德绑架。约束：回复简短（<60字），附带破防指数(0-100)。

### 模块二：妈妈银行存单（辅助功能 - 玩梗/工具）

- **场景：** 调侃“压岁钱妈妈帮你存着”的童年谎言。
- **评分：** 趣味性 ★★★★ | 开发难度 ★
- **功能逻辑：**
  1.  **表单录入：** 姓名、收缴年份、原始金额、收缴理由（如下发买房、给你娶媳妇）。
  2.  **通胀计算（彩蛋）：** 内置计算公式。例如：`金额 * (腾讯/茅台股价涨幅)`。
  3.  **生成凭证：** Canvas 绘制一张复古风“银行定期存单”或“欠条”。
  4.  **导出：** 保存图片到相册。

### 模块三：马年开运头像（辅助功能 - 氛围）

- **场景：** 更换春节限定头像。
- **评分：** 刚需度 ★★★ | 开发难度 ★★
- **功能逻辑：**
  1.  **上传/获取：** `wx.chooseImage` 或获取微信头像。
  2.  **贴纸编辑：** 提供“马头套”、“财神帽”、“赛博功德光环”等透明 PNG 素材。
  3.  **合成导出：** Canvas 合成 500x500px 图片。

## 3. 技术架构方案

鉴于开发者为资深后端，推荐采用 **Serverless** 架构以减少运维成本，加快上线速度。

### 3.1 技术栈选型

- **前端框架：** **UniApp (Vue3 + Vite)**
  - _优势：_ 一套代码编译小程序，开发效率高，生态组件（uView/ThorUI）丰富。
- **后端服务：** **微信小程序云开发 (WeChat CloudBase)**
  - _优势：_ 免域名备案、免HTTPS证书、免服务器运维、原生鉴权。
- **AI 模型层：** **DeepSeek-V3 API**
  - _对接方式：_ 通过云函数 (Cloud Functions) 转发调用，隐藏 API Key。

### 3.2 关键接口定义

1.  **Chat (云函数):**
    - Input: `{ message: string, role_id: string }`
    - Process: User Input -> 敏感词检测(`msgSecCheck`) -> DeepSeek API -> Output。
    - Output: `{ reply: string, score: number }`
2.  **Security (云函数):**
    - 所有 UGC 内容（用户输入的聊天、存单上的文字）必须经过微信官方内容安全接口检测。

### 3.3 成本预估

- **服务器：** 云开发按量付费（首月约 ¥19.9 或免费额度）。
- **AI API：** DeepSeek API 费用（预计 ¥50-100 可支撑数万次调用）。
- **总预算：** < ¥200 (不含人工)。

## 6. 风险控制 (Safety Check)

1.  **内容合规（最高优先级）：**
    - AI 输出必须经过过滤。如果 DeepSeek 返回敏感内容，后端拦截并返回通用兜底文案（如：“二姨去打麻将了，稍后再说”）。
2.  **类目审核：**
    - 如果有“算命”、“运势”字眼容易被拒，统一话术为“娱乐”、“性格测试”、“祝福生成”。
3.  **用户隐私：**
    - 不保存用户上传的照片，所有图片处理均在本地或内存中完成，处理完即销毁（或仅返回临时路径）。
